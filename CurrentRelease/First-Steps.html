First steps

Installation

  • Unzip the downloaded files in any folder you like. If you want to, insert the files from the SampleDB folder, so you can get a feeling of the software right away

First Startup

  • Windows warns you from running the application, as it has no certificate. If you do not trust the exe file I supply, you can build the program on your own. Just download the soure code for that purpose.

Working with the software

  • Get a Pocket Query from geocaching.com.
  • Import the Pocket Query and set the Database files in case you use the supplied ones. (You find the buttons on the Overview Page)
  • It is a good idea to start off the importing of the RouterDb right now. In order to do that, you have to get an OSM pbf file from geofabrik.de, and then import it. This file contains the information the program needs to create routes for you.
  • If you don’t use the provided Databases, you’ll find the Ratingprofiles page empty. Go ahead and give points to what is important to you. If you use the provided Databases you’ll find some presets, feel free to manipulate them. Don’t forget to save!
  • Now you’re ready to click the rate geocaches button in the map window. Some magic should happen and you see the Geocaches in different color, based on your rating.
  • By now the importing of the OSM Data should be done, otherwise please be patient.
  • Go to the Routingprofiles tab now. Again, if you’ve used the provided databases, you’ll find some presets. Feel free to change them.
  • Now select a startpoint (and an endpoint) from the map by rightclicking anywhere on the map or by writing the coordinates into the appropriate fields. Click ‘Create Route’ and after some moments you should see the perfect route for your next geocaching tour ;)

I hope the rest is self-explanatory, however there are a few tips and tricks listed here

Have fun!
